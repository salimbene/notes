# Conceptos

## Criterio

- Esta carpeta reune notas tematicas que no giran alrededor de un unico autor.
- Incluye corrientes, conceptos filosoficos, teologia y notas tecnico-conceptuales.
- Las obras y textos puntuales quedan fuera de esta carpeta para una pasada posterior.
- Los nombres de archivo se normalizan en minusculas y con guiones.
